# Memory Kernel

A minimal substrate that enables physical objects, places, and living things to acquire verifiable digital presence through human encounter.

The kernel is not an application. It is the foundation that applications are built upon — private memory spaces for gifted vinyl records, digital presences for physical books, presence practices around trees.

## Primitives

- **Anchor** — bind an identifier to a physical subject
- **Witness** — signed attestation of encounter
- **Identifier** — label that refers to a subject, accumulating knowledge over time
- **Relationship** — verifiable connection between identifiers
- **Scope** — visibility boundary governing access

## Architecture

Built on Ethereum (Celo L2) using:

- **ERC-8004** — identity and reputation registries for anchors and witnesses
- **MetaMask Delegation Framework** — scoped permissions via caveat enforcers
- **ENS** — human-readable identifier resolution via wildcard resolver
- **Self Agent ID** — ZK proof-of-humanity for privacy-preserving witnessing
